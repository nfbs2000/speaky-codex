Builder Takeaway

좋은 파일 인덱스는 단순한 목록이 아니라 "질문 -> 경로" 맵이어야 합니다. 자신의 에이전트 프로젝트에도 같은 방식의 인덱스를 만들어 두면, 새 팀원이 저장소를 훨씬 빨리 이해할 수 있습니다.
